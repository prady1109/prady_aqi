About

About RSquare Evaluation

This evaluation metrics tell us about how well the model has fitted on the training dataset i.e. how well the model has learned the relationship between the independent variables and the dependent variable.

Learn more

About RMSE Evaluation

Root Mean Square Error method of evaluation is a well reknowned one and is used everywhere to calvulate the error between the predicted and the actual values. Here Rmse is calculated on 5-folds i.e. using K-fold cross validation.

Learn more

About the Dataset

The dataset for traing and testing the models has benn taken from Central Pollution Control Board website. The dataset has independent features which are: Temperature, Humidity, Wind-Direction, Wind-Speed, Month, Previous Day PM2.5 Concenteration, Solar Radiation and Pressure. The independent feature which has to be predicted is the next day PM2.5 concenterate.
